## docker-node-canvas
Docker image with dependencies installed for building project with [node-canvas](https://github.com/Automattic/node-canvas)

## Environment

* Ubuntu 14.04 (Trusty)
* node version: 6.3.1
* npm version: 3.10.3
* yarn version: 0.23.4

## Usage
```
$ docker pull cettoana/node-canvas
```

or use in `Dockerfile`
```
FROM cettoana/node-canvas

RUN ...
```
